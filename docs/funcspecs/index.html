Catalina Functional Specifications

이 문서 영역은 톰캣의 Catalina 서블릿 컨테이너 부분에서 지원하는 여러 기능에 대한 functional specificationg을 포함합니다. 대부분의 경우 이러한 기능이 기본 서블릿이나 JSP 명세에 문서화되지 않은, 예상 올바른 행동의 정의는 이러한 기능 사용자에 모두 중요하므로, 테스트할지 결정하려고 작가를 테스트하기.

기능적 사양은 메뉴 (왼쪽)에 다음과 같은 범주로 나누어집니다 :

  • 사용 관리 애플 리케이션 - 툴을 통해 Tomcat은 설치뿐만 아니라, 자신의 툴에 대한 자세한 요구 사항을 구성하고 운영하는 능력을 지원하기위한 전반적 요구 사항.
  • 사용 내부 서블릿 - 내부, 컨테이너 - 관리, 서블릿으로 구현됩니다 Catalina 기능에 대한 요구 사항.
  • 사용 영역 구현은 - org.apache.catalina.Realm 을 인터페이스의 구현을 위한 요구 사항 (사용자, 암호 및 역할의 컬렉션에 대한 액세스를 제공하는) 그는 표준 톰캣배포에 포함됩니다.

NOTE - 어떤 경우에는 이러한 기능 사양의 내용이 기존 구현에서 "엔지니어링 리버스 '이었습니다. 이 운동은 왜 Catalina 에 소개가 제공되어야하는지, 확실한 방법없이 성취 할 수 없기 때문입니다.
